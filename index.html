i18nT

angular translation module with custom filters for different languages with support for different grammar cases

Live example

This is example of using i18nT

{{'PL' | i18nT}} {{'SHOW_HELLO' | i18nT}}

{{'POSTS.PERSON_POSTS' | i18nT:[{gender:'male',n:post,k:post-2,first:'Peter',second:'John'}]}}
{{'POSTS.PERSON_POSTS' | i18nT:[{gender:'female',n:post,k:post-2,first:'Anna',second:'Christine'}]}}

index simple case male case female case
{{post}} {{'POSTS.INFO' | i18nT:[{n:post}]}} {{'POSTS.PERSON_POSTS' | i18nT:[{gender:'male',n:post,k:post-2,first:'Peter',second:'John'}]}} {{'POSTS.PERSON_POSTS' | i18nT:[{gender:'female',n:post,k:post-2,first:'Anna',second:'Christine'}]}}
